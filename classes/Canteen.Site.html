Canteen.Site Class

Extends Canteen.EventDispatcher
Namespace: Canteen

The Canteen Site is the client wrapper for the server API. Here's an example of how to initialize on your site:

$(function(){
    var site = new Canteen.Site({
        contentId : "#content",
        pageTitleId : "#pageTitle"
    });
    site.on('ready', function(){
        // Site is ready!
    });
});
